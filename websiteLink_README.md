# Lizzie Soltis Personal Website

https://lizziesoltis.netlify.app

This repository contains the source code for my personal website, showcasing my portfolio, skills, and experience.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

You need a web browser and a local server to run this website. If you don't have a local server, you can use Python's built-in HTTP server or any other local server of your choice.

### Installing

1. Clone the repository
   ```
   git clone https://github.com/LizzieSoltis/website.git
   ```
2. Navigate to the project directory
   ```
   cd website
   ```

### Running the website
#### Off MacOS:

1. Navigate to the website repository folder in finder.
2. Double click on index.html

#### Using Python's built-in HTTP server:

1. If you have Python installed, you can use its built-in HTTP server. In the project directory, run:

   For Python 3.x:
   ```
   python -m http.server
   ```

   For Python 2.x:
   ```
   python -m SimpleHTTPServer
   ```

2. Open your web browser and go to `http://localhost:8000`
